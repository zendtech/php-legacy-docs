pht\Queue::push

(PECL pht >= 0.0.1)

pht\Queue::push — Pushes a value to the end of a queue

Description

public pht\Queue::push ( mixed $value ) : void

This method will add a value onto the queue.

Parameters

value

The value to be added to a pht\Queue. This value will be serialised (since it may be passed around between threads).

Return Values

No return value.

Examples

Example #1 Pushing a value to a queue

<?php

use pht\Queue;

$queue = new Queue();

$queue->push(1);

var_dump($queue);

The above example will output:

object(pht\Queue)#1 (1) {
  [0]=>
  int(1)
}

fann_set_learning_rate — Sets the learning rate

Description

fann_set_learning_rate ( resource $ann , float $learning_rate ) : bool

Sets the learning rate.

More info available in fann_get_learning_rate().

Parameters

ann

Neural network resource.

The learning rate.

Generator::valid

(PHP 5 >= 5.5.0, PHP 7)

Generator::valid — Check if the iterator has been closed

Description

public Generator::valid ( void ) : bool

Parameters

This function has no parameters.

Return Values

Returns FALSE if the iterator has been closed. Otherwise returns TRUE.

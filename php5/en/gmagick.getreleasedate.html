Gmagick::getreleasedate — Returns the GraphicsMagick release date as a string

Description

public Gmagick::getreleasedate ( void ) : string

Returns the GraphicsMagick release date as a string.

Parameters

This function has no parameters.

Return Values

Returns the GraphicsMagick release date as a string.

Errors/Exceptions

Imagick::mapImage

(PECL imagick 2.0.0)

Imagick::mapImage — Replaces the colors of an image with the closest color from a reference image

Warning

This function has been DEPRECATED as of Imagick 3.4.4. Relying on this function is highly discouraged.

Description

Imagick::mapImage ( Imagick $map , bool $dither ) : bool

Parameters

map

dither

Return Values

Returns TRUE on success.

Errors/Exceptions

Throws ImagickException on error.

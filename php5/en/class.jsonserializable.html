The JsonSerializable interface

(PHP 5 >= 5.4.0, PHP 7)

Introduction

Objects implementing JsonSerializable can customize their JSON representation when encoded with json_encode().

Interface synopsis

JsonSerializable {
/* Methods */
abstract public jsonSerialize ( void ) : mixed
}

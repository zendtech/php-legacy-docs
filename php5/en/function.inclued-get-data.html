inclued_get_data

(PECL inclued >= 0.1.0)

inclued_get_data — Get the inclued data

Description

inclued_get_data ( void ) : array

Get the inclued data.

Parameters

This function has no parameters.

Return Values

The inclued data.

Examples

Example #1 inclued_get_data() example

See the inclued examples section for ways to create graphs with this data.

<?php 
include 'x.php';

$clue = inclued_get_data();

print_r($clue);
?>

The above example will output something similar to:

Array
(

      [0] => Array
        (

          [op_type] => 2
          [filename] => x.php
          [opened_path] => /tmp/x.php
          [fromfile] => /tmp/z.php
          [fromline] => 2
        )
    )
)

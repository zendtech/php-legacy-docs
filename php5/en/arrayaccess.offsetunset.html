ArrayAccess::offsetUnset

(PHP 5, PHP 7)

ArrayAccess::offsetUnset — Unset an offset

Description

abstract public ArrayAccess::offsetUnset ( mixed $offset ) : void

Unsets an offset.

Note:

This method will not be called when type-casting to (unset)

Parameters

offset

The offset to unset.

Return Values

No value is returned.

PDF_scale

(PHP 4, PECL pdflib >= 1.0.0)

PDF_scale — Scale coordinate system

Description

PDF_scale ( resource $p , float $sx , float $sy ) : bool

Scales the coordinate system. Returns TRUE on success or FALSE on failure.

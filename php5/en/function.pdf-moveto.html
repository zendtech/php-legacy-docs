PDF_moveto

(PHP 4, PECL pdflib >= 1.0.0)

PDF_moveto — Set current point

Description

PDF_moveto ( resource $p , float $x , float $y ) : bool

Sets the current point for graphics output. Returns TRUE on success or FALSE on failure.

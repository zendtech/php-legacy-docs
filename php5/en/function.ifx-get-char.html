ifx_get_char

(PHP 4, PHP 5 < 5.2.1)

ifx_get_char — Return the content of the char object

Description

ifx_get_char ( int $bid ) : string

Returns the content of the char object.

Parameters

bid

The char object-id.

Return Values

Returns the contents as a string, or FALSE on errors.

Judy::offsetSet

(PECL judy >= 0.1.1)

Judy::offsetSet — Offset to set

Description

public Judy::offsetSet ( mixed $offset , mixed $value ) : bool

Assigns a value to the specified offset.

Parameters

offset

The offset to assign the value to.

value

The value to set.

Return Values

No value is returned.

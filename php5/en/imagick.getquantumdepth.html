Imagick::getQuantumDepth

(PECL imagick 2.0.0)

Imagick::getQuantumDepth — Gets the quantum depth

Description

Imagick::getQuantumDepth ( void ) : array

Returns the Imagick quantum depth as a string.

Return Values

Returns the Imagick quantum depth as a string.

Errors/Exceptions

Throws ImagickException on error.

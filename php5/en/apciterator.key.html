APCIterator::key

(PECL apc >= 3.1.1)

APCIterator::key — Get iterator key

Description

public APCIterator::key ( void ) : string

Gets the current iterator key.

Parameters

This function has no parameters.

Return Values

Returns the key on success, or FALSE upon failure.

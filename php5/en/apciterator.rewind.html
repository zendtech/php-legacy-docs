APCIterator::rewind

(PECL apc >= 3.1.1)

APCIterator::rewind — Rewinds iterator

Description

public APCIterator::rewind ( void ) : void

Rewinds back the iterator to the first element.

Parameters

This function has no parameters.

Return Values

No value is returned.

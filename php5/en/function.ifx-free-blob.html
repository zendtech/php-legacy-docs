ifx_free_blob

(PHP 4, PHP 5 < 5.2.1)

ifx_free_blob — Deletes the blob object

Description

ifx_free_blob ( int $bid ) : bool

Deletes the blobobject for the given blob object-id.

Parameters

bid

The BLOB object id.

Return Values

Returns TRUE on success or FALSE on failure.

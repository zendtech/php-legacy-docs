PDF_lineto

(PHP 4, PECL pdflib >= 1.0.0)

PDF_lineto — Draw a line

Description

PDF_lineto ( resource $p , float $x , float $y ) : bool

Draws a line from the current point to another point. Returns TRUE on success or FALSE on failure.

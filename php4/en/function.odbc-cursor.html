odbc_cursor

(PHP 4, PHP 5)

odbc_cursor — Get cursorname

Description

string odbc_cursor ( resource $result_id )

Gets the cursorname for the given result_id.

Parameters

result_id

The result identifier.

Return Values

Returns the cursor name, as a string.

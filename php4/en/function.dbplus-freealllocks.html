dbplus_freealllocks

(PHP 4 <= 4.1.0, PECL dbplus >= 0.9)

dbplus_freealllocks — Free all locks held by this client

Description

int dbplus_freealllocks ( void )

Frees all tuple locks held by this client.

Return Values

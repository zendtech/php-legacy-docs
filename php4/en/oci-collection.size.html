OCI-Collection::size

(PHP 5, PECL OCI8 >= 1.1.0)

OCI-Collection::size — Returns size of the collection

Description

int OCI-Collection::size ( void )

Returns the size of the collection.

Return Values

Returns the number of elements in the collection or FALSE on error.
